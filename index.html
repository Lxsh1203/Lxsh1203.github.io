1、安装及初步配置

1.1 安装

    （Windows请点击最左侧的蓝色按钮，Linux为中间按钮，Mac为右侧，预览版在左下角）

打开后，同意协议。

点击下一步，建议都勾选。（后面几个涉及到环境变量、右键菜单快捷打开、与打开cpp文件有关）
安装即可。

安装之后打开vscode，（我这里安装了汉化和很多插件，所以左侧栏和语言不一样，正常首次安装应该是英文）应该如下图所示。

1.2汉化及插件下载

点击上面图片左侧栏从上往下数第五个图标，打开拓展页面（你之后会经常用上它，因为vscode本来就是靠插件获取功能的）。

在搜索框里输入Chinese，点击第一个，然后点击install安装。然后点击弹出来的消息按钮（如果你不是第一次安装，且之前开过消息免打扰，就看不到，要在消息页面查看）。

2.2打开vscode

win+s键搜索Developer，点击cmd或powershell。（最好用管理员，否则可能报错，也可以设置此应用默认以管理员打开）

在这个界面，输入cd /d 你想用的盘符:(例如cd /d D:)
但有个很出生的东西就是，cmd要加/d，powershell不需要，否则要么没用，要么报错
使用cd命令打开你要的文件夹（接下来cmd和pwshell就都不要/d），例如cd Lxsh1203（文件夹名称）
或者用mkdir Lxsh1203创建一个文件夹（只需要创建一次，之后就省略这一步），之后再用cd打开
然后输入 code 并回车。

这是vscode就被打开啦！（要用msvc就必须这样打开，但请定期普通打开，因为管理员打开无法更新vscode）

VPN

前言（本文主要参考https://waimao.pro/pack/）

近期，AI已经进入了编程界，各种大模型层出不穷，可令人遗憾的是，许多模型只能在国外使用，例如chatgpt等等。但一切困难都不能阻挡程序 猿 员！

本博客就来教大家如何使用ai！

购买

官方网址：今天要用到的网站，打开应如图所示。

这里点击右上角的English，然后更改成中文。

点击注册。这里会有一个cloudflare的人机验证，等待即可，如果过不去，就从隐身模式里进入网站（缺点是每次都要登录）。
接下来进入如下界面。

填写个人信息并成功注册。
注册之后会让你验证邮件地址。（弹窗）

网站会向邮箱发一个邮件，里面有两个链接，第一个是官网（外国），第二个是镜像网站（国内），打开第二个链接。
在这个网页登录（如果你换了设备，比如用电脑注册，但用手机看邮件）或者等待，就会看到绿色对勾提示你验证完毕。
点击左上角服务，然后点订购新服务，进入如下界面。根据自身需求选择月或者年。注意设备数量限制，如果是团队要看清楚再订购。

选择后进入付款页面。

点击继续。

记得填入JMS9272283，可以得到优惠！然后点击收款处。确认信息后选择支付宝（alipay），扫码付款。

建议大家选择1年的套餐，优惠更划算，也不用经常续费，老牌的厂商，可以比较放心

使用

我已经购买过了，为了截图有注册了一个账号，接下来我换成之前的账号带大家看接下来的步骤。

点击服务、我的服务。此时会有一个可用服务。

点开它。在这个页面你可以升级套餐、退款等。

然后先放下这个网站，来安装应用。
Windows电脑：v2ray
Mac电脑：v2ray梅开二度
IOS手机：potasto
Android手机/华为：v2ray没有打广子，只是这个好用

Windows

打开v2rayN.exe（一个字都不要差），然后在系统右下角缩小应用中找到它。（红色的V）

这时回到网站，点击第4个DNS links+QR。

你可以通过扫描二维码、复制链接添加服务器。（右键缩小的软件）

这里将系统代理改为自动。路由改为绕过大陆，这样就不用手动配置代理，并且在访问国内网站时也不会浪费流量（一个月有流量限制，但一般500G够用）。
但如果你要把它当加速器用或者软件并没有得到加速时就切换成全局，例如我玩花园战争2时就会用这个。
之后每次打开v2rayn的时候，就会自动连接VPN啦。

Mac、IOS、Android

因为本人没有此类设备或者懒得弄（IOS），所以暂时没有。可以在https://waimao.pro/pack/里找。
